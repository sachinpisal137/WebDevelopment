Command-Line Task Instructions

Task Description

This task involves creating a new directory and navigating into it using the command line.

Instructions

Step 1: Create a New Directory

To create a new directory named my_project, use the following command:

mkdir my_project

Step 2: Navigate into the New Directory

To navigate into the newly created directory, use the cd command:

cd my_project

Step 3: Verify Current Directory

To verify that you are in the correct directory, use the pwd command:

pwd

Congratulations! You have successfully completed the task.
